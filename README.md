# Game of Life

Educational team project to implement Conway’s Game of Life in C.

## Requirements & Implementation

- Terminal-based visualization using ASCII graphics
- Field size: **80×25** with wrap-around edges (toroidal grid)
- Interactive mode:
  - `A` / `Z` — increase / decrease speed
  - `Space` — exit the program
- Initialization via `stdin`
- At least **5 predefined patterns** for quick start
- Structured programming style:
  - Functions ≤ 42 lines
- Memory-safe implementation (no leaks)
- Code must pass:
  - `cppcheck`
  - `clang-format`
- No use of `system()` or similar system calls
- Optional: use of `ncurses` for handling input/output
